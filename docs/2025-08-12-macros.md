## Module: `scenarioAnalysis` (VBA/scenarioAnalysis)

Handles running Old/New scenario comparisons and exporting results to a separate workbook.

| Macro/Function | Type | Description |
|---|---|---|
| `getOutputSheets()` | Private Function | Returns an array of the four output sheet names. |
| `getScenarios()` | Private Function | Builds a dictionary defining two scenarios ("Old" and "New") with cell-level parameter mappings on the standard and tailored test input sheets. |
| `initializeOutputs()` | Sub | Main entry point. Creates a new workbook, iterates through each scenario, sets parameters, forces recalculation, copies output sheets, and builds a table of contents with hyperlinks. |
| `printOutputs(owb, scenario)` | Function | Copies each output sheet into the target workbook, renames with scenario suffix, breaks links, returns new sheet names. |
| `breakLinks(owb)` | Sub | Breaks all external Excel links in the given workbook. |

```vba
'----------------------------------------
'SET UP TEMPLATE/SCENARIO VARIABLES
'----------------------------------------

Private Function getOutputSheets() As Variant
 getOutputSheets = Array( _
        "Output 1-1 - External DSA", _
        "Output 1-2 - Public DSA", _
        "Output 2-1 Stress_Charts_Ex", _
        "Output 2-2 Stress_Charts_Pub" _
    )
End Function

Private Function getScenarios() As Scripting.Dictionary
    SCENARIO_SHEET_1 = "Input 6(optional)-Standard Test"
    SCENARIO_SHEET_2 = "Input 6 - Tailored Tests"

    Set getScenarios = New Scripting.Dictionary
    getScenarios.Add "names", Array("Old", "New")
    getScenarios.Add "scenarios", New Scripting.Dictionary
    With getScenarios.Item("scenarios")
        .Add SCENARIO_SHEET_1, New Scripting.Dictionary
        With .Item(SCENARIO_SHEET_1)
            .Add "C4", New Scripting.Dictionary
            With .Item("C4")
                .Add "Old", "Old"
                .Add "New", "New"
            End With
            .Add "C8", New Scripting.Dictionary
            With .Item("C8")
                .Add "Old", "Off"
                .Add "New", "On"
            End With
        End With

        .Add SCENARIO_SHEET_2, New Scripting.Dictionary
        With .Item(SCENARIO_SHEET_2)
             .Add "C6", New Scripting.Dictionary
            With .Item("C6")
                .Add "Old", "Off"
                .Add "New", "On"
            End With
        End With
    End With
End Function

'----------------------------------------
'Run scenarios
'----------------------------------------

Sub initializeOutputs()
    Dim owb As Workbook, scenarios As Scripting.Dictionary, tblContents As Worksheet, outputSheets As Variant
    Dim scenarioLog As Range
    Set thiswb = ThisWorkbook
    Set owb = Application.Workbooks.Add
    Set tblContents = owb.Sheets(1)
    Set scenarioLog = tblContents.Range("A1")
    Set scenarios = getScenarios
    listOfScenarios = scenarios.Item("names")
    scenarioId = 0
    For Each scenario In listOfScenarios
        scenarioId = scenarioId + 1
        scenarioLog.Value = "Scenario " & scenario

        For Each sName In scenarios.Item("scenarios").Keys
            Set scenarioSheet = thiswb.Sheets(sName)
            If scenarioSheet Is Nothing Then
                MsgBox "Can't find sheet " & sName
                Exit Sub
            End If
            With scenarioSheet
                For Each addr In scenarios.Item("scenarios").Item(sName).Keys
                    If scenarios.Item("scenarios").Item(sName).Item(addr).Exists(scenario) Then
                        .Range(addr).Value = scenarios.Item("scenarios").Item(sName).Item(addr).Item(scenario)
                    End If
                Next addr
            End With
        Next sName

        Application.Calculate
        outputSheets = printOutputs(owb, scenarioId)
        rCount = 0
        For i = 1 To UBound(outputSheets)
            scenarioLog.Offset(rCount + 1, 1).Value = outputSheets(i)
            scenarioLog.Offset(rCount + 1, 1).Hyperlinks.Add Anchor:=scenarioLog.Offset(rCount + 1, 1), Address:="", SubAddress:= _
               "'" & outputSheets(i) & "'!A1", TextToDisplay:= _
               outputSheets(i)
            rCount = rCount + 1
        Next i

        Set scenarioLog = scenarioLog.Offset(rCount + 1)

    Next scenario
    tblContents.Activate
    tblContents.Name = "Table of Contents"
End Sub

Function printOutputs(ByRef owb As Workbook, ByVal scenario As String) As Variant
    Dim outputSheets As Variant, outputSheetNames As Variant
    ReDim outputSheetNames(0 To 0) As Variant
    outputSheets = getOutputSheets
    For Each sh In outputSheets
        With ThisWorkbook.Sheets(sh)
            .Copy Before:=owb.Sheets(1)
            Set ws = owb.Worksheets(sh)
            ws.Name = sh & "(" & scenario & ")"
            ReDim Preserve outputSheetNames(0 To UBound(outputSheetNames) + 1)
            outputSheetNames(UBound(outputSheetNames)) = ws.Name
            breakLinks owb
        End With
    Next sh
    printOutputs = outputSheetNames
End Function

Sub breakLinks(ByRef owb As Workbook)
If Not IsEmpty(owb.LinkSources) Then
    For Each TrueLink In owb.LinkSources
         owb.BreakLink Name:=TrueLink, Type:=xlExcelLinks
    Next TrueLink
End If
End Sub
```

## Module: `downloadMod` (VBA/downloadMod)

Handles downloading and merging country-specific historical data files into the template.

| Macro/Function | Type | Description |
|---|---|---|
| `refreshHistorical()` | Sub | Triggers a country data refresh by calling `onCountryChange` with the currently selected country. |
| `onCountryChange(cCountry)` | Sub | Main data import routine. Opens or downloads the historical data file, copies worksheets into the template (replacing existing ones), preserves named ranges, and breaks links. |
| `openSaveAs(wb, initial)` | Function | Displays a Save As dialog for .xlsm files. Returns `True` if saved, `False` if cancelled. |
| `downloadTo(initial)` | Function | Displays a Save As dialog and returns the chosen file path, or empty string if cancelled. |
| `DownloadFile(file, savename)` | Sub | Downloads a file from a URL via `Microsoft.XMLHTTP` and saves to disk via `ADODB.Stream`. |

```vba
Private Const DSF_NAME_PREFIX = "DSF__"
```

## Summary

The workbook contains **9 VBA modules** (6 standard modules, 1 class module, and 2 sheet/workbook modules with no code). The macros serve these main purposes:
